Problem Statement & Purpose

⚡ Project Objective

The primary objective of this project is not to design a boost converter. The boost converter serves only as an active operational platform for hardware-in-the-loop validation.

To estimate the Equivalent Series Resistance (ESR) of the output capacitor online, without disconnecting the capacitor, without injecting external test signals, and without using expensive laboratory measurement equipment.

🔴 Limitations of Traditional Methods

Conventional ESR measurement methods (LCR Meter, ESR Meter, Impedance Analyzer) require:

✗ Converter shutdown & system downtime
✗ Physical capacitor disconnection
✗ Dedicated offline measurement equipment
✗ Cannot predict failures during operation
✗ Cannot provide real-time health information

Estimation Theory & Core Formulae

Switching Dynamics

MOSFET ON State
  • · Inductor stores energy from input source
  • · Diode blocks backflow from output
  • · Capacitor supplies load current
  • · Output voltage slopes downward
MOSFET OFF State
  • · Inductor releases stored energy
  • · Current forced through diode
  • · Capacitor charges simultaneously
  • · Output voltage rises

Ripple Decomposition

Total Ripple Identity
Total Ripple = Capacitive Ripple + ESR Ripple
Capacitive Ripple

Slow linear ramp — governed by current integration (Q = C·V). Gradual rise/fall between switching events.

ESR Ripple ← Target Signal

Sharp instantaneous voltage step (ΔVESR) at switching boundaries. Directly encodes ESR magnitude.

Key Mathematical Relationships

Core ESR Formula
ESR = ΔVESR / ΔIC
ΔVESR = ESR-induced voltage step
ΔIC = Capacitor current step
Estimator Equation
ESR = (K · |Vadc|) / (GainAmp · ICAP)
K = 10 (calibration factor)
GainAmp = 10
ICAP = 0.3 A
LPF Transfer Function
H(s) = ωc / (s + ωc)
ωc = 1.54 × 10⁵ rad/s
fc ≈ 24.5 kHz cutoff

Equivalent Time Sampling (ETS) Principle

The ripple amplitude is extremely small (typically only a few millivolts). Direct ADC sampling cannot accurately reconstruct the waveform because cost-effective ADCs lack the bandwidth for sub-microsecond voltage steps within a single switching period. ETS overcomes this by collecting samples from many switching cycles and reconstructing a high-resolution ripple waveform over an extended time base.

ETS Reconstruction Concept
Cycle 1→ Sample Point 1
Cycle 2→ Sample Point 2
Cycle 3→ Sample Point 3
···
Cycle N→ Sample Point N
→ Complete ripple waveform reconstructed
ETS Digital Block Implementation
Zero-Order Hold (ZOH) — ADC latching at 4×10⁻⁶ s
Unit Delay (z⁻¹) — Stores previous discrete sample
Difference Block — ΔV(k) = V(k) − V(k−1)
03

8-Stage Signal Processing Pipeline

Because the target ESR step voltage is miniscule (often only a few millivolts) and obscured by high-frequency switching noise, the raw output voltage VC(t) is processed through eight specialized operational blocks.

1
Capacitor Ripple Voltage Measurement
Output: VC(t)

Voltage across output filter capacitor is tracked continuously. Signal carries heavy DC baseline alongside fragile AC ripple component and high-frequency switching noise.

2
Differential Voltage Measurement
Output: Vdiff

Transformer-modeled sensing circuit with balanced resistor divider networks eliminates the dominant DC output voltage component. Extracts ESR-related ripple while rejecting large DC rail.

3
Amplification Stage 1
Gain₁ = R7 / R1 · Output: Vgain1

Isolated ripple passed through initial hardware amplifier stage. Rescues millivolt-level signal from the native noise floor and improves measurement sensitivity.

4
DC Offset Addition (+1.5 V)
Output: Voffset

Fixed +1.5 V DC translation offset added. Shifts waveform into strictly positive voltage window (0 V to 3 V) ensuring safe compatibility with standard unipolar microcontroller ADCs.

5
Amplification Stage 2
Gain₂ = R10 / R8 · Output: Vgain2

Second gain stage scales the voltage profile across the maximum ADC input span, boosting signal-to-noise ratio and enhancing ripple step visibility for accurate estimation.

6
Low-Pass Filtering
H(s) = 1.54×10⁵ / (s + 1.54×10⁵) · Output: VLPF

First-order continuous-time LPF attenuates high-frequency parasitic switching spikes and EMI while safeguarding fundamental ESR phase-step information.

7
Equivalent Time Sampling (ETS)
ZOH + Unit Delay + Difference · Output: ΔVETS

ZOH latches at 4×10⁻⁶ s. Unit Delay stores previous sample. Difference operation: ΔV(k) = V(k) − V(k−1) captures cycle-to-cycle ESR-related voltage changes.

8
MATLAB Function ESR Estimator
Output: ESREstimated

MATLAB Function Block computes ESR via: ESR = (K·|Vadc|)/(GainAmp·ICAP). Includes first-order digital low-pass moving filter for output stability.

Complete Signal Chain
[Boost Converter] → VC(t) → [Differential Extraction] → Vdiff → [Amp Stage 1] → [+1.5V Offset] → [Amp Stage 2] → Vgain
→ [LPF H(s)] → VLPF → [ZOH · z⁻¹ · Δ] → ΔVETS → [MATLAB ESR Estimator] → ESRout

MATLAB Function — ESR Estimator

Exact algorithmic implementation running inside the simulation's digital estimation block. Extracted directly from the project documentation.

ESR_Estimator.m — MATLAB Function Block MATLAB 
function ESR = ESR_Estimator(V_adc, Gain_Amp)

% Persistent variable to retain filter history across simulation steps
persistent ESR_filt

if isempty(ESR_filt)
    ESR_filt = 0;
end

% Operating Setup Constants
I_CAP_STEP = 0.3;   % Configured capacitor ripple current step (Amperes)
K          = 10;    % System empirical calibration scaling factor

% Instantaneous ESR Calculation
% Maps raw digital ETS differential input directly to resistance
ESR_inst = (K * abs(V_adc)) / (Gain_Amp * I_CAP_STEP);

% First-Order Digital Low-Pass Filtering for Smoothing
alpha    = 0.05;   % Smoothing coefficient (IIR filter)
ESR_filt = alpha * ESR_inst + (1 - alpha) * ESR_filt;

% Final Stabilized ESR Output
ESR = ESR_filt;

end

Technical Conclusion

Final Project Statement

The project develops an online capacitor health monitoring system for a boost converter by estimating the output capacitor ESR from output voltage ripple using ripple extraction and equivalent time sampling techniques without disconnecting the capacitor from the converter.

As the estimated ESR curve closely tracks physical component changes under variable load currents and shifting duty cycles, the model is fully validated for deployment in predictive hardware maintenance kits. The system is structured as a continuous-to-discrete signal processing pipeline completely compatible with embedded ADC architectures and low-cost microcontrollers.
